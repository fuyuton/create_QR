１．PNG画像でのダウンロードができること。
２．Nukoを受信できるようなイメージとセットにすること。
３．著作権表示の代わりに使うために、名前を表示できること。
４．Chrome/Firefox/safariで動くこと。
５．そーすをGithubにあげてくれること。
６．できればクライアントだけで完結すること。
